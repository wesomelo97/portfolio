# Projeto Website Pessoal - Ewerton Melo

## Descrição
Este projeto é um website pessoal para Ewerton Melo, um desenvolvedor web. O site serve como um portfólio online, destacando as habilidades, especialidades e projetos de Ewerton.

## Estrutura do Projeto
O projeto consiste nos seguintes arquivos principais:

- `index.html`: A estrutura principal do site.
- `css/style.css`: Estilos CSS para o layout e design do site.
- `js/app.js`: JavaScript para o formulário de contato.
- `js/mobile.js`: JavaScript para o menu mobile.

## Funcionalidades

### 1. Layout Responsivo
O site é totalmente responsivo, adaptando-se a diferentes tamanhos de tela.

### 2. Seções Principais
- **Início**: Uma introdução com uma chamada para ação.
- **Especialidades**: Destaca as áreas de expertise de Ewerton.
- **Sobre**: Uma breve biografia e links para redes sociais.
- **Portfólio**: Showcase de projetos (atualmente com placeholders).
- **Contato**: Um formulário para entrar em contato.

### 3. Menu Mobile
Um menu hamburguer para dispositivos móveis, implementado com JavaScript.

### 4. Formulário de Contato
Um formulário funcional que envia mensagens para o email especificado, utilizando o serviço FormSubmit.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- [Bootstrap Icons](https://icons.getbootstrap.com/)
- [FormSubmit](https://formsubmit.co/) para processamento do formulário

## Como Usar

1. Clone este repositório para sua máquina local.
2. Abra o arquivo `index.html` em um navegador web para visualizar o site.
3. Para desenvolvimento, você pode usar um servidor local como o Live Server do VS Code.

## Personalização

- Atualize as informações pessoais no arquivo `index.html`.
- Modifique os estilos em `css/style.css` para alterar o design.
- Adicione seus próprios projetos na seção de portfólio.

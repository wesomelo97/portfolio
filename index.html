Criatividade e inovação andam lado a lado. Com uma combinação unica de design impactante, funcionalidade intuitiva e otimização para resultados, estou pronto para criar a presença online dos seus sonhos.

MINHAS ESPECIALIDADES.

Website

Seja landing pages, blogs ou outras soluções web, tento criar algo personalizado que atenda às necessidades específicas de cada cliente, utilizando HTML, CSS e JavaScript para garantir interfaces responsivas e atraentes, sempre focando na experiência do usuário e na estética.

Sistema de gestão

Desenvolvo dashboards e sistemas de gestão personalizados que atendem às necessidades específicas de cada cliente. Utilizo C#, .NET e MySQL para criar soluções com back-end confiável e interfaces limpas e funcionais. Meus sistemas são projetados para otimizar processos e facilitar o controle de pequenos negócios.

Loja online

Crio lojas online, sistemas para restaurantes e cafeterias, utilizando ReactJS para desenvolver interfaces de usuário intuitivas e responsivas. Com .NET, gerencio a lógica de negócios e, com MySQL, mantenho dados de produtos e transações. Integro APIs para métodos de pagamento e sistemas de inventário, garantindo uma experiência de compra fluida e segura.
